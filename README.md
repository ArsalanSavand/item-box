# Item Box

Simple app to add and manage any kind of items.

## Features

* Items with name, image and more.
* Random items.
* Fully responsive.
* Naviage with `Backspace` and refresh with `R`.

## Commands

### Run app

`$ npm install`

`$ bower intsall`
 
`$ grunt serve`

### Build app

`$ grunt build`

### Code quality

`$ grunt jshint`

## How to

### How to add items

Modify the `app/assets/data/items.json` file.

### How to add random

Modify the `app/assets/data/randoms.json` file.

## Built with

* AngularJS
* GruntJS
* Bootstrap
* Bootswatch
